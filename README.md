🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.
Play the classic 2-player game directly in your browser!

🔗 Live Demo:
https://tic-tac-toe-aditya-gade.vercel.app/

📌 Features

🎯 2 Player Game (Player X vs Player O)

🧠 Win detection logic

🔄 Restart game functionality

📱 Responsive design

⚡ Lightweight and fast

🛠️ Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

📂 How to Clone This Repository
1️⃣ Open your terminal
2️⃣ Run the following command:
git clone https://github.com/AdityaGade28/Tic-Tac-Toe.git
3️⃣ Go to the project folder:
cd Tic-Tac-Toe
4️⃣ Open the project:

Just open the index.html file in any browser.

🎮 How to Play

Player X starts first.

Click on any empty box to make your move.

The first player to align 3 symbols (row, column, or diagonal) wins.

Click Restart to play again.
